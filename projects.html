Crypto Search and Wallet

The ideal behind this application was to to give the user the capabilty to search crypto and utilize a wallet to consolidate crypto-currency that are held on different platforms. As a crypto-currency investor, one of the main issues i've dealt with is having to manage multiple platforms to keep track of crypto that I possess.

This project is a react-redux based application that utilizes an api call, hooks and class components and a global storage for saving data. It consists of 3 pages: search, form, and wallet page. The search page gives you the ability to search crypto currency within the top 100 parameter. The form page allows a user to input four specifications before submitting the information. The wallet recieves the details from the form and displays the information.

FX Masters Academy

The foreign exchange (also known as FX or forex) market is a global marketplace for exchanging national currencies. Because of the worldwide reach of trade, commerce, and finance, forex markets tend to be the largest and most liquid asset markets in the world. FX Masters Academy is an online school for users to learn about Forex trading

This project is a express node based application that utilized sequelize to save information to a database. This application has login and registration capabilites for users to sign-in or create an account. The sign-in component utilizes bcrypt and passport to authenticate users to gain different levels of access.
